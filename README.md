Meus Estudos de Cibersegurança
Este repositório é o espaço onde registro meus conhecimentos teóricos sobre cibersegurança e redes, desenvolvidos através do programa Mulher Digital e cursos da Cisco. O objetivo de organizar este material aqui é criar uma base de consulta própria e facilitar a revisão dos conceitos fundamentais que estou aprendendo, garantindo que eu possa consolidar esse conhecimento conforme avanço na minha formação em ADS.

Nesta primeira etapa, estou documentando os conteúdos do módulo de Introdução à Cibersegurança, detalhando os conceitos teóricos e as práticas que compõem a base da segurança da informação.
